# Terraform_AWS
Terraform AWS Infrastructure Project

🚀 Terraform AWS Infrastructure Project

Este projeto demonstra a construção de infraestrutura na AWS utilizando Terraform, aplicando boas práticas de Infraestrutura como Código (IaC).

🎯 Objetivo

Criar um ambiente escalável, reutilizável e seguro utilizando Terraform, com foco em aprendizado prático e aplicação profissional.
